# Customer Portal (Future)

This app will provide a customer-facing portal for rental customers to:
- View their rental history
- Access invoices and payment receipts
- Manage their profile
- View upcoming bookings

**Status**: Placeholder - Not yet implemented

/{tenant-slug}/client-dashboard/{customer-id}/payments → Payment history
